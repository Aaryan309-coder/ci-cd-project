# CI/CD Pipeline using GitHub Actions and Docker

## Aim
To implement CI/CD pipeline for application deployment.

## Description
This project demonstrates Continuous Integration and Continuous Deployment using GitHub Actions. A Flask backend application is tested and deployed using Docker.

## Tools Used
- Python (Flask)
- Docker
- GitHub Actions
- Pytest

## Steps Performed
1. Created Flask application
2. Wrote test cases
3. Created Dockerfile
4. Built Docker image
5. Ran Docker container
6. Created CI/CD pipeline using GitHub Actions

## Output
- Application running successfully
- Tests passed
- Docker container running
- CI/CD pipeline executed successfully
